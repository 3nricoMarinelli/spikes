# Time-Series Forecasting with Spiking Neural Network

Spiking neural networks time-series forecasting sandbox.

## Structure

- `forecasting.ipynb` - Main workflow notebook
- `src/` - Source code modules
  - `dataset/` - Data loading
  - `visualization/` - Plotting and visualization
  - `models/` - Neural network architectures and training
  - `evaluation/` - Evaluation metrics
  - `utils/` - General utilities
- `configs/` - Configuration files
- `data/` - Data directory
- `models/` - Saved model checkpoints
- `outputs/` - Experiment outputs

## Executable code
Execution code is in `forecasting.ipynb`
